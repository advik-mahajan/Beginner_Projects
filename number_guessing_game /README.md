NUMBER GUESSING GAME - 
A command-line number guessing game built with Python where the user tries to a randomly generated number within a set range.

FEATURES:

CORE GAMEPLAY:
1) Random Number Generation (between 1 and 100, inclusive)
2) Unlimited Guess Attempts
3) Higher/Lower Feedback after each Guess
4) Attempt Counter Displayed on Winning

INPUT VALIDATION:
The game includes error handling for:
1) Invalid Numeric Input
2) Out-of-Range Guesses

TECHNOLOGIES USED:
1) Python 3
2) built-in random module

FUTURE IMPROVEMENTS:
1) Limited Number of Attempts (with a "Game Over" state)
2) Difficulty Levels (by adjusting the range or attempt limit)
3) Play Again Option
4) High Score / Best Attempt Tracker
