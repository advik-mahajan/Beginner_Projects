ATM SIMULATOR - A Command-Line ATM Simulator built with Python where the user first authenticates with a PIN and can withdraw or deposit money from a simulated account balance.

FEATURES:
CORE FUNCTIONALITY:

1. PIN Authenticity (4-digit PIN, 3 attempts before account lockout) - Enter PIN 4444 to use this code
2. Withdraw Money (with sufficient funds check)
3. Deposit Money
4. EMI Calculator (calculates monthly EMI, total payment, and total interest for a loan)
5. Real-Time Balance Updates Displayed after Each Transaction
6. Cancel Option (enter 0 to cancel a withdraw/deposit and return to menu)
7. Continuous Menu Loop (perform multiple transactions in one session)

INPUT VALIDATION:
THE SIMULATOR INCLUDES ERROR HANDLING FOR:

1. Invalid Numeric Input (non-numeric PIN or amounts)
2. Out-of-Range PIN (must be 4-digit)
3. Negative or Zero Transaction Amounts
4. Withdrawal Amounts Exceeding Balance

TECHNOLOGIES USED:

1. Python 3

FUTURE IMPROVEMENTS:

1. Multiple User Accounts (separate PINs and balances)
2. Transaction History Log
