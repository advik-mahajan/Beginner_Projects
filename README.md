CALCULATOR CLI - 
A command-line calculator built with Python that performs both basic and complex mathematical operations.

FEATURES:

Basic Operations:
1) Addition
2) Subtraction
3) Multiplication
4) Division
5) Exponentiation
6) Remainder (Modulo)
7) Percentage Calculation
Advance Operations:
1) Square Root Calculator
2) Factorial
3) Average of Multiple Numbers
4) Maximum from a List of Numbers
5) Minimum from a List of Numbers

INPUT VALIDATION:
The calculator includes error handling for:
1) Invalid Numeric Inputs
2) Division by Zero
3) Negative Values where Mathematically Invalid

TECHNOLOGIES USED:
1) Python 3
2) built in math module

FUTURE IMPROVEMENTS:
1) History of Calculations
2) Scientific Calculator (eg - Trigonometric, Logarithmic Calculations)
3) Unit Conversions
